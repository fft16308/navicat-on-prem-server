## 核心功能，说人话版

### 数据不出公司，安全不妥协

这是很多企业用 On-Prem Server 而不用 Navicat Cloud 的核心原因。所有的同步和协作流量，都在你的内网走——没有第三方经手，审计起来也清晰。

### 团队管理不麻烦

支持 LDAP / AD 对接，公司已有的账号体系直接用，不用再建一套新的用户管理。管理员可以按人、按组设置访问权限和存储规则，大团队也能管得住。

### 与 Navicat 客户端无缝协作

如果团队本来就在用 Navicat 系列，那集成几乎是零成本的——客户端连上 On-Prem Server，大家的查询、模型、连接配置就能实时共享和同步。

### 网页管理界面，运维不头疼

有内置的网页端管理界面，服务器状态、项目文件一目了然，不用每次都远程登录服务器操作。支持短信和邮件通知，服务器出问题能第一时间知道。
